Text Commands

The following IRC text commands are supported as described. Additional text command will be added in the future.

Alt-B Bars, show channel and PM as bars
Alt-C Channel Panels (Show/Hide)
Alt-H Help, IRC commands (Show/Hide)
Alt-I IRC Controls (Show, collapse, hide)
Alt-L List of IRC Servers (Show/Hide)
Alt-N Next, Cycle IRC channel panels
Alt-P Private Message Panels (Show/Hide)
Alt-S Server Messages (Show/Hide)
Alt-X Hide all panels.
Alt-Z Zoom, (Channel panel only)
