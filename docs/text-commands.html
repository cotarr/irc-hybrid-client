Text Commands

The following IRC text commands are supported as described. Additional text command will be added in the future.

ADMIN /ADMIN [server]
AWAY /AWAY [away-message]
CTCP /CTCP <nickname> <ctcp-command>
DEOP /DEOP <nick1> ... [nick5]
DEVOICE /DEVOICE <nick1> ... [nick5]
JOIN /JOIN <#channel>
LIST /LIST [#channel-mask]
ME /ME <action-message>
MODE /MODE <#channel> [channel-mode(s)]
MODE /MODE <nickname> [user-mode(s)]
MOTD /MOTD [server]
MSG /MSG <nickname> <private-message>
NICK /NICK <new-nickname>
NOP /NOP ...inert testing command...
NOTICE /NOTICE <nickname> <notice-message>
OP /OP <nick1> ... [nick5]
PART /PART <#channel> [Optional part message]
QUERY /QUERY <nickname> <private-message>
QUIT /QUIT [optional quit message]
QUOTE /QUOTE <IRC server raw command>
TOPIC /TOPIC <#channel> <new-topic>
VERSION /VERSION [server]
VOICE /VOICE <nick1> ... [nick5]
WHO /WHO [mask] [o]
WHOIS /WHOIS <nickname>
