Server Rotation Group Number
Server Definition Label

A server group defines a set of IRC server definitions where different IRC servers can be rotated in sequence after disconnect from IRC.

  • The Group input field accepts integer values 0 to 9999
  • Group 0 is used to define individual IRC server addresses that do not rotate.
  • Group 1 or greater defines a set of server where servers of the same group number will automatically rotate different server addresses after disconnect.
  • In typical use, a server group would allow different IRC networks such as DALnet or Undernet to have different group numbers.
